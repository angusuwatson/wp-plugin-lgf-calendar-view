# LGF Calendar View

A WordPress plugin to display Motopress Hotel Booking data in a LibreOffice Calc-style layout.

## Features

- Displays booking data in a table format
- Simple and clean interface

## Installation

1. Upload the plugin files to the `/wp-content/plugins/wp-plugin-lgf-calendar-view` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[lgf_calendar_view]` to display the booking view.
